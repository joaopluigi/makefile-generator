# Makefile Generator

Simple and easy makefile generator made in Python!

### Usage:           
  - set execution permission
  - run makefile_generator.py
  - make
  - binary file named 'program' will be created at ./bin/
  
### !Important!
  - .c files must be on ./src/
  - .h files must be on ./inc/                 
  - libs must be on ./lib/
